Spotting terms

The Spotter performs linguistic preprocessing: tokenization, POS tagging and lemmatization. It also detects the occurrences of single-word terms (SWT) and multi-word terms (MWT). The text files (.txt files) are found in an input directory and the processed files (.xmi files) are stored in an output directory.

The Spotter needs TreeTagger to be installed by the TermSuite user to process De, En, Es, Fr, Ru, Zh. TreeTagger performs POS tagging and lemmatization.

Fill the various parameters, click on the Run button on the top left of this window to launch the processing. Click on the Save button on the top right of this window if you wish to keep this parameter configuration for a next run.
